## ИИ и технологии

### Модели и исследования

- OpenAI закрывает ИИ-браузер Atlas (9 августа сервис будет отключен, функционал переносится в десктопный ChatGPT). Компания сочла создание безопасного браузера слишком сложным и дорогим; вместо отдельных приложений будет развивать ИИ в существующих продуктах — [docsllm](https://t.me/docsllm/79533)

- Новый дайджест: OpenAI представила GPT-Live с одновременным слушанием и ответом, xAI выпустила Grok 4.5 с упором на программирование и агентные задачи, Сбер выпустил GigaChat 3.5 Ultra (open source, MIT-лицензия) с улучшениями в коде и контексте — [neural_houses](https://t.me/neural_houses/1887)

### Продукты и инструменты

- Метод papercuts для агентов: разработчик придумал способ, при котором агенты фиксируют мелкие проблемы в процессе работы (неудачные вызовы инструментов, устаревшую документацию, проблемы настройки). По записям можно улучшать рабочие процессы; на GitHub есть готовый CLI-тул — [tips_ai](https://t.me/tips_ai/4799)

- Обсуждение использования Claude в IDE (VS Code): участники сообщают об интеграции Claude Code с VS Code, обмене опытом лимитов токенов и эффективности разных моделей (Claude Opus, Sonnet, Composer). На примере одного пользователя: использует Claude Code + VS Code с февраля-марта из РФ, аккаунт жив, связывает это с одной американской картой для оплаты — [docsllm](https://t.me/docsllm/79590)

### Применение в юридической практике

- Обсуждение рисков при использовании Claude Code в IDE из России: есть опасения, что Claude может получить доступ к данным через VS Code (таймзона, местоположение) и заблокировать аккаунт. Рекомендация: следить за IP-адресом, если он постоянно меняется географически — высокий риск бана. Советуют использовать батник для подмены локали и временной зоны при запуске VS Code — [docsllm](https://t.me/docsllm/79545)

### Тренды

Поток обновлений в LLM ускоряется: OpenAI и xAI переносят фокус на агентные системы и интеграцию ИИ в существующие инструменты (IDEs, браузеры), а не на отдельные приложения. Одновременно растет интерес к методам оптимизации работы агентов и стандартизации взаимодействия с IDE-плагинами. Для российских пользователей остается вопрос совместимости и риски блокировки при работе с международными AI-сервисами.
