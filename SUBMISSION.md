Файл скилла подготовлен как `radar-dev-SKILL.md`. Скопируй его в папку скилла:

```bash

```

Сделай три скриншота работающего проекта и положи их в папку `images/`:

## 4. Публикация через GitHub CLI

Репозиторий уже существует: `advocatalexy-cell/TG_Radar`. Просто закоммить и запушь новые файлы:

```bash
git add PROJECT.md README.md SUBMISSION.md .claude/skills/radar-dev/ images/
git commit -m "docs: описание проекта, переработанный README, скилл radar-dev, скриншоты"
git push
```

Если репозиторий еще не создан – создать и опубликовать одной командой из папки проекта:

```bash
gh repo create TG_Radar --public --source=. --remote=origin --push \
  --description "Мультиагентный мониторинг Telegram-каналов про ИИ в праве с автопубликацией дайджеста"
```

Проверка: открой ссылку репозитория в режиме инкогнито и убедись, что README и скриншоты отображаются.

## 5. Анкета для публикации выпускного проекта

**1. Название проекта**

Radar – мультиагентный мониторинг Telegram-каналов про ИИ в праве

**2. Краткое описание проекта**

Radar собирает сообщения из ~85 Telegram-каналов по теме ИИ и legal-tech, отсеивает шум через OpenAI и силами нескольких агентов готовит структурированный дайджест, который автоматически публикуется в Telegram-канал. Проект полезен практикующим юристам, руководителям юридических департаментов и ведущим тематических каналов. На выходе – готовая дневная выжимка ключевых сигналов со ссылками на первоисточники вместо часа ручного чтения ленты.

**3. Ссылка на скилл проекта**

[Загрузи файл `.claude/skills/radar-dev/SKILL.md` (или папку `radar-dev`) на Яндекс Диск / Google Диск, открой доступ по ссылке и вставь ссылку сюда.]

**4. Для кого может быть полезен проект**

Для аналитиков; для личного использования; другое – для практикующих юристов и руководителей юридических департаментов.

**5. Что умеет проект**

- собирает сообщения из ~85 Telegram-каналов по MTProto с дедупликацией;
- отфильтровывает шум и оставляет сигналы по ИИ и legal-tech через OpenAI;
- анализирует сигналы двумя специализированными агентами (право и ИИ-технологии);
- собирает единый структурированный дайджест агентом-композитором;
- автоматически публикует дайджест в Telegram-канал по расписанию;
- адаптирует новость или фрагмент дайджеста в готовые посты под Telegram и Bitrix24 (скилл `post-adapter`).
