---

Запусти мультиагентный анализ дайджеста Radar. Следуй точно этому алгоритму.

## Шаг 0: Подготовка

Определи дату: сегодня в формате YYYY-MM-DD.
Проверь, что существуют файлы `data/processed/YYYY-MM-DD-*-processed.json`.
Если файлов нет — сообщи пользователю и остановись.

## Шаг 1: Параллельный анализ

Запусти **одновременно** два субагента через Task:

**Задание для analyst-legal:**
> Проанализируй правовые сигналы за ДАТА.
> Прочитай все файлы data/processed/ДАТА-*-processed.json.
> Отбери сигналы с category=legal/law/regulation или ключевыми словами правовой тематики.
> Сохрани результат в data/analysis/ДАТА-legal.md согласно своей инструкции.

**Задание для analyst-aitech:**
> Проанализируй AI и технологические сигналы за ДАТА.
> Прочитай все файлы data/processed/ДАТА-*-processed.json.
> Отбери сигналы с category=ai/tech или ключевыми словами AI-тематики.
> Сохрани результат в data/analysis/ДАТА-aitech.md согласно своей инструкции.

Дождись завершения обоих агентов.

## Шаг 2: Сборка дайджеста

После завершения шага 1 запусти **последовательно** digest-composer:

**Задание для digest-composer:**
> Собери итоговый дайджест за ДАТА.
> Прочитай data/analysis/ДАТА-legal.md и data/analysis/ДАТА-aitech.md.
> Выдели 3-5 главных событий суммарно.
> Сохрани результат в digests/ДАТА-digest.md согласно своей инструкции.

## Шаг 3: Публикация в Telegram

После успешного сохранения дайджеста запусти публикацию через Bash:

```
python agents/digest-agent.py --send-only --date ДАТА
```

Если переменные окружения `TELEGRAM_BOT_TOKEN` или `TELEGRAM_CHAT_ID` не заданы — не останавливайся, сообщи об этом в финальном отчёте и продолжай.

## Шаг 4: Финальный отчет

Сообщи пользователю:
- Дату обработки
- Сколько сигналов нашел analyst-legal
- Сколько сигналов нашел analyst-aitech
- Путь к итоговому дайджесту
- Статус публикации в Telegram (опубликован / ошибка / переменные не заданы)
- Время выполнения (если известно)

Пример: "Анализ за 2026-06-20 завершен. Правовых сигналов: 4, AI/Tech сигналов: 7. Дайджест: digests/2026-06-20-digest.md. Опубликован в Telegram."

## Контракт данных между агентами

Processed JSON → analyst-* → data/analysis/*.md → digest-composer → digests/*.md → Telegram

Структура processed.json (для справки):
```json
{
  "id": 12345,
  "channel_id": "allthingslegal",
  "date": "2026-06-20T10:00:00+00:00",
  "text": "полный текст поста",
  "classification": {
    "signal": true,
    "score": 0.85,
    "category": "legal",
    "summary": "Краткое описание",
    "keywords": ["закон", "суд"]
  }
}
```
